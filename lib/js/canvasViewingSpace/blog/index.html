2016-08-08-a - Automove Script Editor
---------------------------------------------------------
- There's now an option to open the Automove Script Editor
  on the main menu, here one currently can get the details
  of the current position and add it to the automove script
  (with only the basic movement details)
  One can also type the number of a script element an view
  it's details (without movement details)
- There's also a button to start the presentation, this 
  removes the mouse interface and sets up the keyboard
  - spacebar -> next frame
  - r -> previous frame (which now uses a nice way of using
                         n's movement details to get to n-1's
                         position)
  - q -> end presentation
- New automoveScript commands for working with the script
